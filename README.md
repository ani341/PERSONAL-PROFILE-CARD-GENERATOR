# 📇 Personal Profile Card Generator

A beginner-friendly Python CLI script that takes basic user input, performs string manipulation and simple arithmetic, and outputs a neatly formatted profile card with personalized details.

## ✨ Features

* **String Formatting:** Automatically converts full names to `Title Case` and hobbies to `Capitalized` format.
* **Initials Generator:** Dynamically extracts and formats first and last name initials (e.g., `J.D.`).
* **Dynamic Year Calculation:** Calculates the user's estimated birth year based on their age.
* **Boolean Checks:** Automatically evaluates if the user's full name qualifies as "long" (over 8 characters).
